# Telegram Чат-Бот "Прогноз Погоди"

Цей бот надає актуальний прогноз погоди для будь-якого населеного пункту, використовуючи API OpenWeatherMap.

## Технології
* Python 3.13
* `python-telegram-bot` (версія 22.5)
* `requests` (версія 2.32.5)
* OpenWeatherMap API
